# ProjectLaunch-ADF

Arbol De Fuego Services - Professional lawn and sprinkler service booking platform with admin dashboard.

## Features

- 🌿 Customer booking wizard (5 steps)
- 📋 Admin dashboard with booking management
- 💰 Invoice generation with custom pricing
- 💳 Payment tracking (online card, cash, venmo, physical card)
- 📄 Receipt generation with Google review request
- 📊 Booking status tracking (invoice sent, payment received)

## Getting Started

Simply open `index.html` in your browser to access:
- 🌿 **Site Tab**: Customer booking interface
- ⚙️ **Admin Tab**: Ricardo's admin dashboard

All data is stored locally in your browser (localStorage).

## Admin Features

- View all bookings in calendar/list view
- Generate invoices for completed jobs
- Track payment status
- Edit individual booking prices (for job variations)
- Manage customer notes and booking status

## Deployment

This is a single-page application. Deploy `index.html` to any web host.

Built with HTML, CSS, and vanilla JavaScript.
